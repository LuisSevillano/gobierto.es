Ponemos a disposición de todo el mundo el código fuente de Gobierto, para que pueda ser instalado, revisado y modificado entre todos.

En github.com/populatetools/gobierto puedes encontrar el código fuente de todas las aplicaciones de Gobierto, e instrucciones de cómo instalarlo y colaborar con el proyecto.

Utiliza las incidencias para reportar fallos o realizar preguntas, así como la guía sobre cómo contribuir para proponer cambios y nuevas funcionalidades.

Todos los detalles

  • Todos los módulos de Gobierto disponibles: comparador de presupuestos, página local de presupuestos, módulo de participación, páginas estáticas...
  • Importa de forma sencilla los datos utilizando los scripts incorporados en el repositorio
  • Fácil de instalar: siguiendo los pasos que te describimos aquí puedes instalar una versión de prueba en menos de una hora.
  • Personalización: al disponer del código fuente, puedes personalizar las características de tu instalación.
  • Integración con analíticas: puedes integrarte con Google Analytics y Mixpanel de forma sencilla, modificando el fichero de configuración.
  • Creación de listas de usuarios para envío de newsletter: si quieres, puedes activar Mailchimp para crear una lista de usuarios y suscribirlos a una newsletter.
  • Colaboración: colabora desarrollando el proyecto para que nos beneficiemos todos del esfuerzo individual.
  • Auditoría de código: al ser código abierto, el código es auditable en cualquier momento por todo el mundo.

Para saber más y descargar el software visita la página de Gobierto en Github
